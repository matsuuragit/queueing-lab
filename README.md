# 待ち行列ラボ — 動く待ち行列 (Queueing Lab)

「待ち行列ラボ」は、待ち行列理論（Queueing Theory）における様々なモデルを視覚的にシミュレーションし、グラフィカルに理解するためのWebアプリケーションです。

単一のHTMLファイル（`index.html`）で構成されており、ブラウザで開くだけで即座に動作します。

## 🛠️ 主な機能と特徴

- **多彩な待ち行列モデルのサポート**
  - **M/M/1**: 単一窓口・ポアソン到着・指数サービス
  - **M/M/c**: 複数窓口（マルチサーバー）
  - **M/M/1/K**: 待合室の容量制限あり
  - **M/M/c/K**: 複数窓口・容量制限あり
  - **M/M/c/c (M/M/c/Loss)**: 即時系（コールセンターなど、溢れた呼は即座に損失）
  - **M/D/1**: 単一窓口・一定サービス（決定論的）
- **インタラクティブなビジュアライゼーション**
  - 到着した客が待ち行列に並び、窓口（サーバー）に送られて処理される様子をリアルタイムのアニメーションで表現。
  - 到着率（$\lambda$）やサービス率（$\mu$）、窓口数（$c$）、許容量（$K$）などのパラメータをリアルタイムにスライダーで変更可能。
- **学術的指標のリアルタイム計算とチャート描画**
  - 平均待ち時間（$W_q$）、平均システム内時間（$W$）、平均待ち人数（$L_q$）、平均システム内人数（$L$）などの指標を理論値とシミュレーション実測値の両方で計算。
  - 状態確率の理論分布と実測分布の比較グラフを表示。
- **ポーズ・速度調整機能**
  - シミュレーションの進行スピードの変更や、一時停止機能。
- **ダークモード風の洗練されたモダンUI**
  - 美しいサイバーパンク・ダークテーマ風のデザインで、非常に見やすく直感的に操作できます。

## 💻 ローカルでの実行方法

1. このリポジトリをクローンするか、`index.html` をローカルPCにダウンロードします。
2. `index.html` をお好みのブラウザ（Google Chrome, Firefox, Microsoft Edge, Safari等）でダブルクリックして開きます。
3. ローカルサーバーを起動する必要はなく、オフラインでも完全に動作します。
